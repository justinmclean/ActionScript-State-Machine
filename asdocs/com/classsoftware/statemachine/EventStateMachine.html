Packagecom.classsoftware.statemachine
Classpublic class EventStateMachine
InheritanceEventStateMachine flash.events.EventDispatcher
Implements IStateMachine, IStateMachineEventListener

Adds the ability for a state machine (of any type) to dispatch events on state changes. The state machine will listen for any event you ask it to (via addActionEventListener or addStateChangeEventListener) and call supplied handlers. You can also use the standard addEventListener to listen for ActionEvents or StateChangeEvents.
